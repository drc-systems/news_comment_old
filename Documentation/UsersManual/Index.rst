For editors
============


Introduction
^^^^^^^^^^^^
To integrate the news comment extension in your system,you need to follows these steps.

First step
^^^^^^^^^^

Install the News System extension and integreate to your system. Prepare news detail page with necessory settings.


Plugin Settings
^^^^^^^^^^^^^^^

Select detail page of news. Click on the New Button to add a new
content element to this page and choose News Comment. Place news comment extension just after news extension.Set the storage page id of News Comment system folder for the record storage.
